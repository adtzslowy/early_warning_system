# Quick Start: Telegram Notifications

3 langkah untuk enable Telegram alerts di akun Anda.

## 1️⃣ Dapatkan Chat ID (2 menit)

**Di Telegram:**
- Cari bot: `@getidsbot`
- Kirim: `/start`
- Copy nomor Chat ID yang muncul

Contoh: `987654321`

## 2️⃣ Simpan ke Profil Anda (1 menit)

**Di aplikasi EWS:**
1. Login → Settings → Profil
2. Scroll ke "Notifikasi Telegram"
3. Paste Chat ID Anda
4. Klik "Simpan Profil"

## 3️⃣ Terima Alerts! ✅

**Mulai sekarang, saat ada perubahan level risiko → notifikasi otomatis masuk ke Telegram Anda**

Format pesan:
```
🚨 ALERT BANJIR ROB

Perangkat: TEST001 - Test Device
Lokasi: Ketapang
Level Risiko: Bahaya
Pesan: Water level melebihi threshold
Waktu: 20 Jul 2026 14:30

🔗 Lihat Detail
```

---

## Troubleshooting

**Notifikasi tidak masuk?**
1. Pastikan Chat ID sudah disimpan (Settings → Profil)
2. Buka Telegram → cek apakah ada pesan dari bot
3. Pastikan device sudah assigned ke akun Anda

**Chat ID tidak tahu?**
1. Buka Telegram
2. Cari `@getidsbot` 
3. Kirim `/start`
4. Lihat Chat ID yang muncul
